Description

Official Bioware DLC causes a few minor bugs in standalone campaigns.

It appears that certain resources which should only extend the OC behave like core resources.

This isn't necessarily a complete list. Please add any you discover.

Stone Prisoner

* Codex entry for character Shale appears as a blank which can never be completed.
* rewards M2DA overrides certain lines (10000 and 10100 for sure, possibly others).

Warden's Keep

* Codex entry

Return to Ostagar

* Codex entry

Workarounds

These bugs don't occur when the DLC in question is disabled. Players might be reluctant to accept that solution, of course.

Spurious codex entries could be suppressed if we had access to the DLC plot files. Unfortunately they are encrypted, so we just have to live with it.

2DA row ID issues can be avoided by using a very high range (even in a standalone campaign table that's intended to replace the OC).
