Данный шаблон позволяет вставлять на страницу спойлеры, блоки со скрытым текстом.

Параметры

Параметр Описание Примечание
title Заголовок спойлера.
titlecolor Цвет шрифта заголовка. По умолчанию - чёрный. Цвет указывается в виде его HEX-кода или HTML-имени. Это удобно, чтобы сделать тёмный заголовок и светлый текст на нём.
headcolor Цвет заголовка. По умолчанию - #B6D1ED. Цвет указывается в виде кода предустановленного цвета, также может указываться в виде HEX-кода или HTML-имени. Предустановлённые: 1 - #F1EBB5, 2 - #D7FFC7, 3 - #E0E0E0.
bodycolor По умолчанию - #F6F6F6. Цвет указывается в виде кода предустановленного цвета, также может указываться в виде HEX-кода или HTML-имени. Предустановлённые: 1 - #FFF8DC.
width Ширина спойлера. По умолчанию - 500pt. Допустимо указание как в процентах, так и в конкретном значении.
align Выравнивание спойлера. По умолчанию отсутствует, то есть - слева. Допустимые значения: left, center, right. В случае, если вы укажете left или right, то текст будет обтекать спойлер.
hint Подсказка. Любое значение параметра выведет подсказку в виде стрелки, направленной вправо. Можно использовать если, например, спойлер очень широкий.
uncollapsed Развёрнутое состояние спойлера. Любое значение, переданное с этим параметром, сделает спойлер развёрнутым при загрузке.
content Текст спойлера. В первой строке в некоторых случаях не работает форматирование, например знак "#" не создаёт нумерованный список, хотя ,' ' ' жирный шрифт ' ' ' работает.

Заготовка

{{ruspoiler
|title       =
|titlecolor  =
|headcolor   =
|bodycolor   =
|width       =
|align       =
|hint        =
|uncollapsed =
|content     =}}

Скопируйте этот код и вставьте на вашу страницу. Обратите внимание: Пустые параметры передавать нельзя, то есть передача в шаблон параметра в виде "headcolor=" обнулит значение цвета заголовка, в результате чего его цвет станет прозрачным, и т.п., поэтому удалите неиспользуемые параметры.
