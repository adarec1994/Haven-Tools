The ABI_base.xls file contains the 2da files that define and integrate all skills, spells, talents and item abilities in the game.

The workbook contains the following worksheets:

  • ABI_Base
  • ability_data
  • ability_effects
  • passive_abilities
  • _tooltip_strings
  • _damage_strings
