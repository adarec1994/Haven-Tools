Эта страница описывает работу следующего набора шаблонов:

  • Template:2da start/ru
  • Template:2da column
  • Template:2da end

Данный набор позволяет достаточно легко и быстро создавать таблицы с описанием полей 2DA. Обратите внимание, что наименование шаблона 2da column хоть и подразумевает создание колонки, но в итоге в таблице с описанием создаёт строку.

При использовании шаблона первая строка, а именно ID, пропускается (добавится автоматически из шаблона 2da start/ru), так как она является общей для всех 2DA. В стартовом шаблоне необходимо только указать название таблицы 2DA, которое будет использовано в качестве названия для таблицы с описанием.

Ниже приведены несколько примеров: 

{{2da start/ru  | ANIM_base}}
{{2da column | LABEL | string | Название модели }}
{{2da column | bHasAppearAnim | int | 0: По умолчанию существо не имеет анимации появления <br/>1: По умолчанию существо имеет анимацию появления}}
{{2da end}}

Produces:

Столбец Тип Описание
ID int (en) Уникальный идентификатор (ключ) строки (диапазоны ID и переопределение строк). Идентификаторы перечисляются от меньшего к большему внутри одного 2DA, хотя и не обязательно по порядку.
LABEL string Название модели
bHasAppearAnim int 0: По умолчанию существо не имеет анимации появления
1: По умолчанию существо имеет анимацию появления

Можно использовать и именованные параметры, например: 

{{2da start/ru
|name        = ANIM_base}}
{{2da column
|name        = LABEL
|type        = string
|description = Название модели
}}
{{2da column
|name        = bHasAppearAnim
|type        = int
|description = 0: По умолчанию существо не имеет анимации появления <br/>1: По умолчанию существо имеет анимацию появления
}}
{{2da end}}
