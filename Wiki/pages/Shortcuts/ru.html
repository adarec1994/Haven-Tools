Скриптинг

В окне скриптов:

  • F1: Общая справка (в отдельном окне)
  • F2: Автозавершение
  • F7: Компиляция
  • CTRL+Space: Автозавершение
  • CTRL+Shift+Space: Подсказка по функции (почти весь список)
  • Двойной щелчок (на функции): Помощь (на панели помощи)

На панели просмотра Функций/Переменных/Констант/Шаблонов:

  • Единичный щелчок (на функции): Помощь
  • Двойной щелчок (на любом элементе): вставить в скрипт
  • Return/Enter (на любом элементе): вставить в скрипт
  • Курсор вправо/вниз: следующая запись в списке
  • Курсор влево/вверх: предыдущая запись в списке
  • Page Up: первая видимая запись
  • Page Down: последняя видимая запись
  • Home: первая запись в списке
  • End: последнаяя запись в списке

Когда открыто окошко автозавершения:

  • Двойной щелчок (на любом элементе): вставить в скрипт
  • Return/Enter (на любом элементе): вставить в скрипт
  • Курсор вправо/вниз: следующая запись в списке
  • Курсор влево/вверх: предыдущая запись в списке
  • Page Up: первая видимая запись
  • Page Down: последняя видимая запись
  • Home: первая запись в списке
  • End: последнаяя запись в списке

Area editor(Редактор области)

Более подпобная информация здесь 3D control.

  • "r": Включить стандартный инструмент выделения (Standard selection tool) (объект должен быть выбран)
  • "q": Включить инструмент перемещения по осям (объект должен быть выбран)
  • "e": Включить инструмент вращения по осям (объект должен быть выбран)
  • "t": Использование локальных координат (объект должен быть выбран)
  • Средняя кнопка мышки: Вращение
  • CTRL: панорама
  • CTRL+Правая кнопка мышки: Вращение
  • Функция ZDrag (Перемещение объектов вверх и вниз): Левая кнопка мышки + Shift + Alt (Предварительно нужно кликнуть на выбранный объект)
  • Двигать камеру: Левая кнопка мышки + Ctrl
  • Rotate Objects Yaw Left: Правая кнопка мышки + Shift

Думаю, для этих команд нужно использовать один из инструментов 3d-перемещения.

  • Rotate Objects Pitch Up: Правая кнопка мышки + Shift + Ctrl
  • Rotate Objects Roll Right: Правая кнопка мышки + Shift + Alt

Cutscene curve editor(Редактор катсцен-кривых)

  • K - Установить все ключи
  • P - Установить позицию
  • O - Установить ориентацию
  • F - FOV
  • Вставить камеру = C
  • Вставить VFX = V
  • Добавить трэк = T
  • Добавить действие:
    • Проиграть анимацию = alt+A
    • (Дополнительные горячие клавиши для других полезных функций)
  • Добавить переход = X
  • Добавить эффект:
    • Появление= I
    • Исчезание= O
  • Редактировать FaceFX = alt+F

Stage Editor(Редактор сцен)

  • Вставить позицию(Place)= P
  • Вставить камеру = C
  • Скрыть объект = H
  • Показать объекты= S

Dialog Editor(Редактор диалогов)

  • Редактировать FaceFX= alt+F
  • Сгенерировать жесты для строчки= alt+G
  • Найти ссылки= alt+L
  • Предпросмотр строки=alt+/
  • Открыть катсцену(Cutscene)= alt+C
  • Открыть сцену(Stage)= alt+S
  • Создать катсцену(cutscene)= alt+ctrl+C
