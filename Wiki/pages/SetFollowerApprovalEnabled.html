Enable the approval rating for a follower

void SetFollowerApprovalEnabled(
object oFollower,
int bEnabled
);
Parameters:
oFollower
follower whose approval of the hero has changed
bEnabled
enabled state
Returns:

Nothing.

Source:

script.ldf

Description

Enabling approval allows the follower's approval rating to be modified and will show the approval widgets in the GUIs. Disabling approval will cause the approval rating to be lost.
