Параметры

Параметр Описание Примечание
title Текст заголовка.
titlealign Выравнивание заголовка. По умолчанию - слева. Возможные значения - left, right, center.
headcolor Цвет заголовка. По умолчанию - #B6D1ED. Цвет указывается в виде кода предустановленного цвета, также может указываться в виде HEX-кода или HTML-имени. Предустановлённые: 1 - #F1EBB5, 2 - #D7FFC7, 3 - #E0E0E0.
bodycolor Цвет тела окна. По умолчанию - #F6F6F6. Цвет указывается в виде кода предустановленного цвета, также может указываться в виде HEX-кода или HTML-имени. Предустановлённые: 1 - #FFF8DC.
width Ширина окна. По умолчанию - 100%. Допустимо указание как в процентах, так и в конкретном значении.
height Высота окна. По умолчанию - по высоте текста. Допустимо указание как в процентах, так и в конкретном значении.
float Плавающее окно. По умолчанию - none. Возможные значения - left, right, none. В случае, если указать left или right, окно будет выравнено по указанной стороне с обтеканием текстом.
clear Расположение других элементов вокруг окна. Возможные значения - left, right, both, none.
content Содержание окна.
bottommenu Нижнее меню. Расположено в правой стороне окна. Передаётся строка со ссылками. Для разделения ссылок используйте шаблон {{mnemonic|1}}.

Чистый шаблон

{{rublock
|title      = 
|titlealign = 
|headcolor  = 
|bodycolor  =
|width      =
|height     =
|float      =
|clear      =
|content    =
|bottommenu =
}}

Скопируйте этот код и вставьте на вашу страницу.

Обратите внимание: Пустые параметры передавать нельзя, то есть передача в шаблон параметра в виде "headcolor=" обнулит значение цвета заголовка, в результате чего его цвет станет прозрачным, и т.п., поэтому удалите неиспользуемые параметры.
