Placeables.xls

From Dragon Age Toolset Wiki
Jump to: navigation, search

The placeables.xls file contains the 2da data that define every placeables' appearance, properties and interactions. These 2da files are used by the Toolset, in the Placeable editor, and by the game engine.

The workbook contains the following worksheets:

  • placeable_basetypes
  • placeable_interactions
  • placeable_types
  • StateCnt_AOE
  • StateCnt_AreaTransition
  • StateCnt_BBase
  • StateCnt_Bodybag
  • StateCnt_Bridge
  • StateCnt_Cage
  • StateCnt_Container
  • StateCnt_Container_Static
  • StateCnt_Door
  • StateCnt_Door_Secret
  • StateCnt_FlipCover
  • StateCnt_Furniture
  • StateCnt_Informational
  • StateCnt_NonSelectable_Trap
  • StateCnt_Puzzle
  • StateCnt_Selectable_Trap
  • StateCnt_Static
  • StateCnt_TopBallista
  • StateCnt_Trap_Trigger
  • StateCnt_Trigger
  • sound_plc_column_names
