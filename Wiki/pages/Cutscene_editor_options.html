Cutscene editor options

From Dragon Age Toolset Wiki
Jump to: navigation, search

The Cutscene editor options are accessible from the options menu.

⊟ General
Default Time Scale Frames (30 FPS) or seconds (100 FPS)
Layout Top to bottom or left to right
Mouse Control Style Windows Standard, NWN, or 3DS Max. See 3D control
Objects Follow Surface
Show Stage
Viewports One, Two (left/right), Two (top/bottom), Three (single bottom), Three (single top) and Four viewports are supported.
⊟ 3D Objects
Bounding Box Active Color
Bounding Box Inactive Color
Orientation arrow inner color
Orientation arrow outer color
Show orientation arrows Always, Never, or Selected
⊟ Camera
Default View Angle
Pitch Maximum
Pitch Minimum
Scaling Zoom
Zoom Distance Maximum
Zoom Distance Minimum
Zoom Speed Fast
Zoom Speed Slow
⊟ Grid
Grid Center Color
Grid Center Radius
Grid Color
Grid Square Size
Grid Squares per Side
⊟ Keys
Ordinal curve colors Define up to ten different colors for the lines show in the curve editor
⊟ Master Object
Master Color
Master Selected Color
Master Size
⊟ Performance
Accurate Headtracking Scrubbing Can also be enabled/disabled with the button in the cutscene toolbar. See Headtracking#Cutscene editor performance for more detail.
Render Objects As Boxes
ROAB Creature Color
ROAB Item Color
ROAB Placeable Color
ROAB Proxy Color
ROAB VFX Color
Update VFX before playing
⊟ Renderer
Enable Lighting
Fade Geometry Cutaway
Fade Geometry Punch Through
Show Pathfinding Points
⊟ Safe Frame
Action Safe Frame Color
Aspect Ratio
Safe Frame Color
Title Safe Frame Color
⊟ Snap Points
Node Color
Node Size
Radius
Selected Node Color
⊟ Target Camera
Target Color
Target Selected Color
Target Size
Target Wireframe
⊟ Timeline Snapping
Snap Events to Events
Snap Events to Frames
Snap Scrubber to Frames
⊟ Trajectory
Line Color
Show Trajectories
Vertex Color
Vertex Selected Color
Vertex Size
