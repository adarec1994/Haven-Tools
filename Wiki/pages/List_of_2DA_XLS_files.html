List of 2DA XLS files

From Dragon Age Toolset Wiki
Jump to: navigation, search

The XLS files listed here can be found under the Toolset's directory [Program Files]\Dragon Age\tools\Source\2DA\.

Bear in mind that 2DAs names are actually based on the names of the worksheets inside these Excel files, not the names of the files themselves. Some of these files contain a large number of individual 2DAs within them. They are bundled together into Excel files based on the nature of the 2DAs for convenience.

XLS files

Generic

  • 2DA base.xls - referenced by several other 2DAs for a variety of purposes
  • AI abilities cond.xls - [Undocumented]
  • AOE.xls - [Undocumented]
  • Ambient ai.xls - [Undocumented]
  • Commands.xls - [Undocumented]
  • Events.xls - [Undocumented]
  • Guitypes.xls - defines the different categories of abilities (talents/spells/hidden)
  • Injuries.xls - handles the effects of injuries.
  • Parts.xls - defines different body parts for use with equipment placing (hip, left hand, upper-right back, etc.)
  • Popups.xls - [Undocumented]
  • Properties.xls - [Undocumented]
  • Ranges.xls - defines the different range scopes in-game (i.e. touch, spell, long)
  • Screenshake.xls - [Undocumented]
  • Shapechange.xls - [Undocumented]
  • Skills.xls - [Undocumented]
  • Summons.xls - defines ranger summons as wolf, bear and spider by default; also defines mage summon as "animated dead"
  • TS System.xls - [Undocumented]
  • Traps.xls - defines all trap types and their effects

Creatures

  • APR_base.xls - appearance types for creatures
  • ABI_base.xls - integrates all talents and spells into the game
  • ANIM_base.xls - defines basic creature animations
  • ANIM_combat.xls - defines creature combat animations
  • ANIM placeables.xls - [Undocumented]
  • background.xls - defines the various origin stories available to each race and class.
  • CLA_base.xls - defines the character classes used throughout the game
  • CLA data.xls - [Undocumented]
  • Creatureranks.xls - [Undocumented]
  • RACE_base.xls - defines the races used throughout the game
  • SyncAnimations.xls - [Undocumented]

Objects

  • BITM base.xls - item types
  • ItemVariations.xls - item appearance variations (eg, all the different ways a longsword can look)
  • Materialtypes.xls - contains stats and other information about the various materials used for scaling
  • placeables.xls - interactive object types
  • PRJ_base.xls - how various projectiles look and behave
  • VFX_base.xls - visual effects

Areas and plots

  • areadata.xls - sets various properties for specific areas
  • Plotactions.xls - [Undocumented]
  • plottypes.xls - how plots are grouped at the highest level in the game GUI (codex entry types, story, etc.)
  • rewards.xls - rewards for plot completion
  • worldmaps.xls - world maps for open-ended travel between areas

Extending the official campaign

  • PRCSCR.xls - "Post-Release Content Script" - specifies scripts to run when an area or area list is loaded or entered. (This file is not included with the game, but is a placeholder for any usermade one)
